### Color Palette for Project

Our application uses a premium, accessible color system based on the official TU Dublin brand guidelines, modernized for a high-end SaaS interface.

#### Brand & Premium Structural Colors
* **Primary Brand (Regal Blue):** `#004C6C`
* **Secondary Accent (Antique Brass):** `#C27D54`
* **App Background Base (Slate-Grey):** `#EEF2F6`
* **Premium Surface White (Alabaster/Pearl):** `#F9FAFB`
* **Text Dark (Charcoal):** `#1E293B`
* **Neutral Silver (Borders):** `#E2E8F0`

#### IT Ticket Status Colors (Semantic Colors)
* **Status: Open (Amber):** `#F59E0B` *(A warmer, more modern yellow)*
* **Status: In Progress (Cyan):** `rgb(104, 202, 248)` *(A bright, professional blue)*
* **Status: Resolved (Emerald):** `#10B981` *(A crisp, premium green)*
* **Action: Delete (Rose/Red):** `#EF4444`
